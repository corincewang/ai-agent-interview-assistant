# AI Agent Interview Assistant

An agentic technical interview coach for SWE, frontend, backend, and AI agent roles.

This repository is currently in the contract-first phase. The code defines method
signatures, schemas, and orchestration boundaries before implementation.

## MVP Scope

- Upload resume, knowledge base, and job description.
- Build a candidate profile.
- Analyze the target role.
- Generate a structured technical interview plan.
- Run a text-first mock interview.
- Evaluate answers and generate a feedback report.

## Architecture

- `backend`: FastAPI, LangGraph, tool and skill contracts.
- `frontend`: Next.js app contracts.
- `docs`: MCP, agent, and product planning notes.
